# How to connect MultiGP and Velocidrone

Its possible to run a race on Velocidrone and manage it via RotorHazard using the RH VELOCIDRONE plugin on Github. This opens up the possibility of running sim races that are fully integrated with the MultiGP race management system.

## Why This Matters

For chapters like UGKL that run regular Cyber Prop sim racing events, this integration means:

- **Automated timing** — RotorHazard handles all lap timing directly from Velocidrone
- **MultiGP integration** — Results sync to MultiGP automatically
- **Professional race management** — Heat assignments, frequency management, and results all in one place

## How to Set It Up

1. Install RotorHazard on your race management computer
2. Download the RH VELOCIDRONE plugin from Github
3. Configure Velocidrone to connect to your RotorHazard instance
4. Set up your MultiGP event through RaceSync
5. Run your race — everything syncs automatically
